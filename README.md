# FITQUEST-360

## PROJECT OBJECTIVE
Build an AI-driven 3D tailoring and gaming ecosystem.

## STRUCTURE
- **backend/**: FastAPI (Python) application handling the Core AI Measurement Engine and Database interactions.
- **frontend/**: React + Three.js Fiber application for the 3D Character Pipeline and Game Frame.

## KEY MODULES
1. **Core AI Measurement Engine**: `backend/ai_engine`
2. **3D Character Pipeline**: `frontend/src/3d_pipeline`
3. **Game Frame**: `frontend/src/game`
4. **ChatBot**: Integration with local LLM.

## SETUP
1. **Backend**:
    ```bash
    cd backend
    pip install -r requirements.txt
    uvicorn main:app --reload
    ```
2. **Frontend**:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
